# Car Parking Management System

A full-stack Car Parking Management System built using MERN Stack.

## Features

- Vehicle Entry
- Vehicle Exit
- Auto Slot Allocation
- Parking Status
- Parking Fee Calculation
- Responsive UI using Tailwind CSS

## For connect Backend to frontend
- cors

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

### Clone Repository

```bash
git clone <your-github-url>
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## API Endpoints

### Vehicle Entry

```http
POST /api/parking/entry
```

### Vehicle Exit

```http
PUT /api/parking/exit/:id
```

### Get Vehicles

```http
GET /api/parking
```
